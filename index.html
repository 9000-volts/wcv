cvdraw is an extensible web painting module.
wcv, this site, is a painting program based off of cvdraw.
It adds an interface that allows you to control the brushes properly, as well as other neat things like ^Z and ^Y keycombo support.
This was coded by 9000 volts as a quick project.
THIS IS AN EXPERIMENT AND IS NOT INTENDED AS AN ALTERNATIVE TO PROPER DRAWING TOOLS!
